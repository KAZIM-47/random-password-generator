# Random Password Generator

A simple, interactive password generator built with HTML, CSS, and JavaScript. Users can select password length and character types (uppercase, lowercase, numbers, symbols) and copy generated passwords easily.

- Real-time password strength indicator
- Copy password to clipboard with tooltip confirmation
- Responsive and visually appealing design

## Usage

1. Open `rpg.html` in your browser.
2. Use the slider to select password length.
3. Check the boxes for the character types you want to include.
4. Click **Generate Password** to create a new password.
5. Click the copy icon to copy the password to clipboard.
